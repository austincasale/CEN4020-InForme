How to run android app:

Needs to be run on Android phone with latest software
Developer options may need to be on

*Copy app-debug.apk from repository into an Android phone SD card
*Explore the files from the phone and find app-debug.apk, click it
*Phone will ask to run the app with special permissions
*Install
*Open

So far it only has two editTexts that posts data to the firebase when the submit button is pushed. If you need to see if data is actually posted, I can add you as an owner if you provide a gmail.
